MD Text Tool is a lightweight web utility for converting Markdown into clean plain text.

MD Text Tool 是一个轻量网页工具，用于将 Markdown 转为纯文本。

It supports import, copy, download, and local history for daily writing workflows.

它支持导入、复制、下载和本地历史，适合日常写作流程。

What this site provides功能列表

  • Markdown to plain text conversionMarkdown 转纯文本
  • .md import via click or drag and drop点击或拖拽导入 .md 文件
  • One-click copy and .txt export一键复制和 .txt 导出
  • Local conversion history本地转换历史
