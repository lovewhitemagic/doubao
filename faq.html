FAQ

What syntax is supported?支持哪些语法？

Headings, lists, links, image alt text, code, emphasis and blockquotes.支持标题、列表、链接、图片替代文本、代码、强调和引用。

Can I import .md files?能导入 .md 文件吗？

Yes, by drag and drop or the import button on converter page.可以，支持拖拽导入和按钮导入。

Is conversion local?转换是否本地进行？

Yes. Conversion runs in your browser, and history is stored in localStorage.是。转换在浏览器本地完成，历史保存在 localStorage。

What encoding is used for downloads?下载文件使用什么编码？

Downloaded .txt files are generated in UTF-8 encoding for broad compatibility.下载的 .txt 文件使用 UTF-8 编码，兼容性更好。

How can I clear conversion history?如何清空转换历史？

Open the History page and use the “Clear History” action. This only clears local browser data.打开历史页面并点击“清空历史”。该操作只会清除当前浏览器本地数据。

Does it support large Markdown files?支持大文件吗？

It works for most common file sizes. Very large files may slow down rendering depending on your device.可处理常见大小文件。超大文件在部分设备上可能出现渲染变慢。

Why does output look slightly different?为什么输出和预期略有差异？

Different Markdown engines support different extensions. This tool keeps a practical common subset.不同 Markdown 引擎支持的扩展语法不同；本工具采用实用的通用子集。

Which browsers are recommended?推荐使用哪些浏览器？

Latest Chrome, Edge, Safari, and Firefox provide the best clipboard and file API compatibility.推荐使用最新版 Chrome、Edge、Safari、Firefox，剪贴板与文件 API 兼容性更好。
